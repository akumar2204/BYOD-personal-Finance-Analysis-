📊 Personal Finance Analytics Report (2025)

🚀 Project Overview
This project transforms one year of self-recorded personal expense data (2025) into a structured financial analytics dashboard using Excel and Power BI.

Instead of treating expenses as simple daily records, this project applies a business intelligence mindset to personal finance. The objective was to convert raw transaction data into measurable KPIs, structured insights, and behavioral analysis.

This is a BYOD (Bring Your Own Data) project — built entirely using my own real financial data.

🎯 Why I Started This Project
I used to record daily expenses in Google Keep without analyzing them.
I realized:
Recording is not equal to understanding.
Raw data without analysis has no decision value.
Spending patterns remain invisible without structured insights.

So I decided to:
Learn Power BI practically
Apply data analysis to real life
Improve financial literacy early
Turn daily habits into measurable performance

🧠 Core Business Question
How can raw personal expense data be transformed into actionable financial intelligence?

This project answers that question.

📊 KPI Framework
The dashboard was built around structured financial KPIs:
🔹 Total Expense - Total annual spending.

🔹 Average Monthly Spend - Baseline monthly financial commitment.

🔹 Total Transactions - Measures spending frequency and behavioral intensity.

🔹 Cash Spend & Digital Spend - Payment allocation analysis.

🔹 Cash % vs Digital % - Financial traceability indicator.

🔹 Category Ranking - Identifies dominant cost centers.

🔹 Highest Spending Month - Detects peak financial exposure.

🔹 Monthly Change % - Measures month-over-month volatility.


Each KPI was designed to answer a behavioral or financial control question.
